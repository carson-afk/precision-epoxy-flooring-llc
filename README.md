# Precision Epoxy Floor LLC

Marketing website for Precision Epoxy Floor LLC, Idaho Falls' flake floor and full-system epoxy specialists.

- **Production**: https://precisionepoxyflooringllc.com
- **Hosting**: Vercel (static)
- **Stack**: vanilla HTML + CSS + JS, no framework

## Local preview

Just open `index.html` in a browser, or run any static server from the repo root.

## Pages

- `/` – Home with estimator + lead form
- `/services` – Service overview
- `/services/residential-garage`, `/services/shop-floor`, `/services/commercial-flake`
- `/gallery`
- `/areas` + 9 city pages (Idaho Falls, Ammon, Iona, Ucon, Rigby, Shelley, Blackfoot, Rexburg, Pocatello)
- `/about`
- `/contact`

## Deploy

Auto-deploys on every push to `main` via Vercel's GitHub integration.
